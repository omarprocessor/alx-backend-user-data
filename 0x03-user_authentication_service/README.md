# User Authentication Service

This project implements a user authentication service using Python Flask, SQLAlchemy, and bcrypt for secure password hashing.

## Requirements

- Python 3.7
- Ubuntu 18.04 LTS
- SQLAlchemy 1.3.x
- bcrypt

## Project Structure

- `models/`: Database models
- `auth/`: Authentication service implementation
- `app.py`: Main Flask application

## Installation

1. Create a virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate
```

2. Install dependencies:
```bash
pip install flask sqlalchemy bcrypt pycodestyle
```

## Usage

Run the application:
```bash
python3 app.py
```
